# 🎭 混乱棋局 (Chaos Chess)

象棋 × 围棋 × 扑克炸弹的创新混合对战游戏

## 🎮 游戏简介

这是一款融合了中国象棋、围棋和扑克炸弹玩法的创新策略游戏。玩家可以在同一棋盘上：
- 移动象棋子进行传统象棋对战
- 下围棋子包围对方的象棋子
- 使用炸弹清除区域内的棋子

## 🚀 快速开始

### 安装依赖
```bash
bun install
# 或使用 npm
npm install
```

### 运行游戏
```bash
bun run dev
# 或使用 npm
npm run dev
```

游戏将在浏览器中自动打开：`http://localhost:9001`

### 构建生产版本
```bash
bun run build
# 或使用 npm
npm run build
```

## 📖 游戏规则

### 🎯 胜利条件
将死对方，或对方将/帅被吃掉/提掉/炸掉

### ♟️ 象棋模式
- 正常象棋规则
- 点击棋子选中，点击目标位置移动
- **可以吃掉围棋子**

### ⚫ 围棋模式
- 在交叉点下围棋子
- **四面包围可提掉象棋子**（上下左右全围住）

### 🔧 围棋战术
- 围棋子可作为**炮架、蹩马腿、挡车**等
- 参与象棋战斗

### 💣 炸弹模式
- 每方2次机会
- 点击中心点炸掉3×3区域内所有棋子（象棋+围棋）
- **将/帅免疫**

### 🔄 回合机制
每回合只能选择一种操作（移动象棋 OR 下围棋 OR 使用炸弹）

## 📝 开发说明

所有游戏逻辑都在 `index.html` 中：
- **CSS**：`<style>` 标签中的样式
- **HTML**：`<body>` 中的页面结构
- **JavaScript**：`<script>` 标签中的游戏逻辑

核心类：`ChaosChessGame`
- 棋盘绘制
- 移动验证
- 包围检测
- 胜负判定
